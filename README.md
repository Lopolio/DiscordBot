# DiscordBot
A small Management Bot for the District Discord Server

## General
This Bot is managed by Lopolio/Sensei/Joe, If you wish to contribute in any way feel free to contact mein the Discord

## Possibilities
Some of the Planned Features

### Roles and Leveling
The Bot will take care of automatic roles considering levels in the Server. Users will gain XP by simply being online.
Ass well by leveling up the User will receive new levels and the Bot will assign these Users new Roles.
The Bot will also Provide new Roles that can be bought by its currency system that has yet to be implemented.
As Well some possibility to warn users will be implemented to the way that when a User is warned he will actually be punished by loosing XP
and also some levels depending on how often he has been punished.

### Anime Related
The Bot will provide several Anime related commands, like requesting a random anime, or receiving some wallpapers

### Polls
A polling mechanism will be added so the someone may start a Poll where Users can add possible answers in the first phase. 
In the second phase Users can vote for these possibilities, except for the possibility that they added themself.
The last phase will consists of displaying the results.
